# ＝LOVE野口衣織　非公式 沼落ちガイド

=LOVE（イコールラブ）のメンバー、野口衣織さんの魅力を凝縮したファンサイトです。
「沼落ち度」に合わせて、彼女の多彩なパフォーマンスやキャラクターを楽しめるコンテンツをまとめています。

## サイトの特徴

### 1. 厳選されたコンテンツ
MV、ライブ映像、歌ってみた、踊ってみた、フォーカス映像など、野口衣織さんの魅力が詰まった動画を厳選して掲載しています。
各動画には、見どころや楽曲の背景解説（ファン目線のコメント）を添えています。

### 2. 気分に合わせたタグ検索
その日の気分に合わせて、動画をフィルタリングできます。
- **#カッコいい**: 憑依的な表現力やキレのあるダンスが見たい時
- **#かわいい**: アイドル全開の笑顔やキュートな振る舞いに癒やされたい時
- **#アイドル**: 王道のキラキラしたパフォーマンスを楽しみたい時
- **#切ない**: 表現力の高さを感じる、感情揺さぶる歌声を聴きたい時

### 3. 沼落ち度メーター
動画を視聴して「視聴済み」マークをつけることで、あなたの「沼落ち度」が上昇します。
視覚的にファンの深度を楽しめる機能です。
