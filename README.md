# 野口衣織 沼落ちガイド

野口衣織さんの魅力を紹介するファンサイトです。
動画（MV、ライブ、歌ってみた、踊ってみた）をカテゴリーやタグでフィルタリングして閲覧できます。

## 技術スタック
- **HTML5**
- **React** (CDN版, Babel standalone)
- **Tailwind CSS** (CDN版)

特別なビルド環境は不要で、ブラウザで `index.html` を開くだけで動作します。

## 公開方法（デプロイ）

このサイトを無料で公開するには、以下の方法が簡単です。

### 方法1：Netlify Drop（おすすめ・一番簡単）
1. PC上で、`index.html` が入っているフォルダを開きます。
2. ブラウザで [Netlify Drop](https://app.netlify.com/drop) にアクセスします。
3. フォルダごと、画面の点線枠内にドラッグ＆ドロップします。
4. 数秒で公開され、URLが発行されます。

### 方法2：GitHub Pages
1. GitHubリポジトリを作成し、`index.html` をアップロードします。
2. リポジトリの **Settings > Pages** に移動します。
3. **Branch** を `master` に設定してSaveします。
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

## サイトの更新方法

PC上でファイルを修正した後、以下の手順で公開サイトを更新できます。

1. フォルダ内の `deploy.bat` をダブルクリックして実行します。
2. 画面の指示に従って変更内容（例:「動画追加」「デザイン修正」など）を入力し、Enterキーを押します。
3. 自動的にGitHubへアップロードされ、数分後に反映されます。
