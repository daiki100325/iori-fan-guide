# GitHub連携の準備ガイド

アンチグラビティ上で更新した内容を、自動的に公開サイト（GitHub Pages）に反映させるためには、**Git（ギット）** というツールが必要です。
現在、あなたの環境にはGitがインストールされていないようなので、以下の手順でセットアップをお願いします。

## 手順1：Gitのインストール

1.  [Git for Windowsのダウンロードページ](https://git-scm.com/download/win) にアクセスします。
2.  「**Click here to download**」をクリックしてインストーラーをダウンロードします。
3.  ダウンロードしたファイル（例: `Git-2.xx.x-64-bit.exe`）を実行します。
4.  インストールウィザードが表示されます。設定項目が多いですが、基本的にすべて **「Next」** を押し続けて大丈夫です（標準設定で問題ありません）。
5.  インストールが完了したら、パソコンを **再起動** してください（環境変数を反映させるため）。

## 手順2：GitHubリポジトリの作成

1.  [GitHub](https://github.com/) にログインします（アカウントがない場合は作成してください）。
2.  右上の「+」アイコンから **「New repository」** を選択します。
3.  **Repository name** に好きな名前を入力します（例: `iori-fan-guide`）。
4.  **Public**（公開）を選択します。
5.  「Add a README file」などは **チェックせずに**、「Create repository」をクリックします。
6.  作成後に表示される画面のURL（例: `https://github.com/YourName/iori-fan-guide.git`）をコピーしておいてください。

---

## 準備ができたら

インストールとリポジトリ作成が完了したら、チャットで **「Gitの準備OK」** と教えてください！
その後、私がコマンドを実行して、現在のプロジェクトをGitHubにアップロードし、自動公開の設定を行います。
